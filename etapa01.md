#  Atividade 2: Escolha do Problema (O Desafio)  
Criação de um software de reserva para uso de laboratórios e empréstimo de equipamentos em universidades.

# Atividade 3: Criação da Visão do Produto e Personas
Usuários finais: Alunos e Colaboradores da universidade.

# Atividade 4: O Product Backlog Inicial (Histórias de Usuário)

1.Como um aluno, quero visualizar o calendário de disponibilidade dos laboratórios para identificar datas e horários livres para reserva.

2.Como um colaborador, quero visualizar o calendário de disponibilidade dos laboratórios para planejar atividades acadêmicas e administrativas.

3.Como um aluno, quero solicitar a reserva de um laboratório para realizar atividades acadêmicas em um horário específico.

4.Como um colaborador, quero solicitar a reserva de laboratórios para realizar aulas, treinamentos, pesquisas ou eventos institucionais.

5.Como um aluno, quero consultar os equipamentos disponíveis para empréstimo para identificar os recursos necessários para minhas atividades.

6.Como um aluno, quero solicitar o empréstimo de equipamentos para utilizá-los em atividades acadêmicas autorizadas.

7.Como um administrador de laboratório, quero visualizar todas as solicitações de reserva do laboratório sob minha responsabilidade para gerenciar a utilização dos recursos disponíveis.

8.Como um administrador de laboratório, quero aprovar solicitações de reserva de laboratórios para autorizar o uso dos espaços de forma controlada.

9.Como um administrador de laboratório, quero rejeitar solicitações de reserva quando houver conflitos ou restrições para garantir o uso adequado do laboratório.

10.Como um administrador de laboratório, quero aprovar solicitações de empréstimo de equipamentos para controlar a disponibilidade e a utilização dos recursos institucionais.

11.Como um administrador de laboratório, quero rejeitar solicitações de empréstimo de equipamentos quando necessário para evitar indisponibilidade ou uso inadequado dos equipamentos.

12.Como um administrador do sistema, quero cadastrar laboratórios e associá-los aos respectivos administradores para organizar a estrutura institucional da universidade.

13.Como um administrador do sistema, quero cadastrar colaboradores como administradores de um ou mais laboratórios para delegar a gestão dos recursos institucionais.
